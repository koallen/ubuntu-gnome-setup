# GNOME Ubuntu setup

## Themes

Install the `numix-icon-theme-circle` theme

```bash
$ sudo apt-add-repository ppa:numix/ppa
$ sudo apt-get update
$ sudo apt-get install numix-icon-theme-circle
```

In `Tweak Tool`, set `GTK+` theme to be `Numix` and `Icons` theme to be `Numix-Circle`.
